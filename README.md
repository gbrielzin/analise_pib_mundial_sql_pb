# 🌍 Análise da Concentração do PIB Global - Projeto End-to-End

Projeto focado em entender a concentração econômica mundial através de um fluxo completo de **ETL**.

## 🔄 1. ETL (PostgreSQL)
Realizei o tratamento da base de dados bruta utilizando **SQL**:
* **Transformação:** Padronização de nomes, tratamento de nulos e tipagem de dados.
* **Lógica:** Filtros de ano e ordenação para identificação do Top 10 mundial.

## 📊 2. Visualização (Power BI & DAX)
* Criação de medidas em **DAX** para cálculo de participação percentual.
* Dashboard focado em **Storytelling** e legibilidade.

## 📈 Insights Gerados
* Identificação de forte dependência global nas duas maiores potências (EUA e China).
* Visualização clara da disparidade entre o Top 2 e o restante do mundo.

## 🛠️ Stack Utilizada
`PostgreSQL` | `Power BI` | `DAX` | `ETL`
